Student Stress Predictor is a machine learning-based web application designed to analyze and predict students' stress levels based on psychological, academic, and social factors. The system uses multiple ML models to classify stress into categories such as Low, Medium, and High, helping users understand their mental health status and take preventive actions.

The application features an interactive user interface where users can input parameters like anxiety level, depression, sleep quality, study load, academic performance, social support, peer pressure, and bullying. Based on these inputs, the trained model predicts the stress level and provides personalized recommendations.

🔹 Key Features
🧠 Machine Learning Models: Logistic Regression, SVM, KNN, Decision Tree, Random Forest, XGBoost
📊 Accurate Predictions: Achieved up to 89.09% accuracy using optimized ensemble models
🎛️ Interactive UI: Slider-based input system for user-friendly data entry
📈 Visual Results: Gauge chart showing stress percentage
📝 Personalized Suggestions: Recommendations based on predicted stress level
📂 History Tracking: Stores past predictions for monitoring mental health trends
📄 PDF Report Generation: Downloadable mental health report
🔹 Tech Stack
Frontend: HTML, CSS, JavaScript (or React + Tailwind CSS)
Backend: Python (Flask)
Machine Learning: Scikit-learn, XGBoost
Visualization: Chart.js / Custom Gauge UI
🔹 How It Works
User inputs stress-related factors via UI
Data is preprocessed and passed to trained ML model
Model predicts stress level (Low / Medium / High)
Results are displayed with percentage and recommendations
Data is stored for history and can be exported as PDF
